# SKILLS.md - Gerenciamento de Habilidades

Este diretório contém habilidades específicas do projeto carregadas via `/.callm/skills`.

## Definição de Skill
Uma skill é uma ferramenta ou conjunto de instruções que estende a capacidade da IA no contexto deste repositório.

## Estrutura sugerida:
- `scripts/`: Scripts executáveis (Python, Node).
- `prompts/`: Diretrizes de comportamento.
- `examples/`: Exemplos de uso.
