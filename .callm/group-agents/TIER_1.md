# TIER_1.md - Grupo de Agentes Triage

Esquadrão de resposta rápida para triagem de bugs e novas ideias.

## 👥 Composição
1. **Architect Agent:** Valida a viabilidade.
2. **Debugger Agent:** Identifica a causa raiz de falhas.
3. **Security Auditor:** Garante que o patch não abra buracos.

## 🔄 Fluxo de Trabalho
- O Arquiteto recebe a demanda.
- O Debugger analisa os logs.
- O Auditor revisa o código sugerido.
- O commit é gerado apenas com 3 aprovações.
