# NEURONS.md - Memória Cognitiva do Projeto

Unidades de conhecimento persistente que o caLLM utiliza para aprender e recordar.

## O que é um Neurônio?
Um "neurônio" é um fragmento de informação técnica ou de negócio que não deve ser esquecido entre as sessões.

## Exemplo:
- Regras de negócio complexas.
- Padrões de banco de dados específicos.
- Histórico de problemas críticos resolvidos.
